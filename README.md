# FIXTOP TRADING P.L.C — Mechanical Tools Landing Page

Static landing page scaffold for FIXTOP TRADING P.L.C using a black and yellow brand palette.

Open `index.html` in a browser, or serve locally with a simple static server:

```bash
python -m http.server 8000
# or with Node.js
npx serve .
```

Files:
- index.html
- css/styles.css
- js/script.js
- images/ (place your local image assets here)
- videos/ (place your local background videos here)

Open `index.html` in a browser or run a local server to view. Images are hotlinked from Unsplash for demo; replace with your assets for production.
